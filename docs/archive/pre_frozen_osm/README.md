# Pre-Frozen OSM Result Archive

This folder holds historical IRP artifacts produced before the OSM travel matrix
was frozen.

Do not use these files as final thesis evidence unless they are explicitly
revalidated under the frozen travel matrix.

For final results, use `docs/results_frozen/`.

## Structure

- `results/`: pre-frozen result JSON/Markdown artifacts, including old headline,
  baseline, sweep, and point-IRP outputs.
- `scripts/`: superseded pre-frozen sweep and baseline scripts that produced
  those historical artifacts.

Matrix-independent forecast/model-selection work is not archived here solely
because it predates the travel-matrix freeze. The archive is for IRP-dependent
numerical evidence that is superseded by frozen reruns or retained only for
audit/history.
