# AI Tool Usage Disclosure

**Project:** ATM Cash Replenishment via Inventory Routing (Master's Thesis)  
**Author:** Onur Soylu  
**Institution:** Otto von Guericke University Magdeburg  
**Date:** 22 May 2026

---

## 1. Purpose

This document provides a short transparency note on the auxiliary use of
generative AI tools during the project. The tools were used as support for
coding, debugging, wording, and research discussion. They were not used as a
substitute for the author's own design decisions, experiments, interpretation,
or academic responsibility.

---

## 2. Tools Used

The following general-purpose AI assistants were used at different points during
the project:

- Claude (Anthropic)
- OpenAI ChatGPT
- Google Gemini

These tools were accessed through standard chat or coding-assistant interfaces.
No custom AI model was trained or deployed as part of this thesis work.

---

## 3. Scope of Assistance

AI assistance was used mainly for:

- discussing alternative implementation approaches;
- improving code structure and documentation clarity;
- debugging and checking code changes;
- drafting or polishing intermediate explanatory text;
- discussing possible interpretations of experimental results;
- suggesting references or concepts for the author to verify independently.

All final decisions about the model structure, experiment design, parameter
selection, thesis argument, and reported interpretation were made by the author.

---

## 4. Author Review and Control

The author reviewed AI-assisted suggestions before using them. Code was run and
tested locally by the author. Empirical results reported in the thesis come from
local experiment runs, not from AI-generated output.

The following parts of the project remain the author's own responsibility:

- problem formulation and research scope;
- dataset handling and understanding;
- selection of the final modelling approach;
- execution of forecast, optimization, simulation, and baseline experiments;
- acceptance or rejection of suggested code/documentation changes;
- interpretation of the results and limitations;
- final thesis writing and academic accountability.

Where AI-assisted text or code suggestions were useful, they were revised,
checked, and integrated under the author's control.

---

## 5. Verification

The repository contains the code, configuration files, experiment scripts, and
selected raw result files needed to inspect the computational work. The main
empirical claims are supported by local run outputs and documented result files,
including the calibration grids and the final 73-day simulation summary.

This disclosure is intended to make the use of auxiliary AI support transparent
while making clear that the thesis work, empirical validation, and final
academic responsibility belong to the author.
